# Crypto Payment Gateway (Web3)

This project simulates a simple crypto payment system.

It helps beginners understand how crypto payments move from sender to receiver.

## What this project does

- Creates payment requests
- Confirms payments
- Stores payment records
- Shows payment history

## Why this project is useful

You will learn:
- How crypto payments work
- How transactions are tracked
- How blockchain confirmations work
- How Web3 apps handle payments

## Tech used

- JavaScript
- Node.js
- Payment concepts
- Blockchain logic (simulated)

## How to run

1. Install Node.js  
2. Clone this repo  
3. Open terminal in project folder  
4. Run: `node app.js`  

## Files overview

- `app.js` → Main app file  
- `payment.js` → Payment logic  
- `confirm.js` → Confirmation logic  
- `storage.js` → Data storage  
- `wallet.js` → Wallet validation  
- `merchant.js` → Merchant logic  
- `config.js` → Settings  
- `utils.js` → Helpers  
- `logger.js` → Logs  
- `package.json` → Project info

## Notes

This project is for learning only.  
Do not use real wallets or funds.
